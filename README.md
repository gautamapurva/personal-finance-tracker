# 💰 Personal Finance Dashboard

An interactive **Personal Finance Dashboard** built using **Python, Dash, Pandas, and Plotly**.
This project helps users **track expenses, analyze spending patterns, and receive financial insights** through a clean dashboard interface.

---

# 🚀 Live Demo

https://personal-finance-tracker-id32.onrender.com



---

# 📌 Features

### 📊 Interactive Dashboard

* Clean and responsive UI built with **Dash**
* Real-time updates when transactions are added

### 💳 Expense Tracking

* Add unlimited transactions
* Store transactions in a **CSV database**

### 📈 Data Visualization

* **Monthly spending bar chart**
* **Category distribution pie chart**

### 📋 Transaction History

* View all past transactions in a **structured table**

### 🧠 Financial Insights

* Automatic **spending analysis**
* Smart **financial advice suggestions**

### 📉 Spending Analytics

* Total spending
* Average expense
* Top spending category

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/your-username/personal-finance-dashboard.git
```

Navigate into the project folder:

```
cd personal-finance-dashboard
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

Open in browser:

```
http://127.0.0.1:8050
```
